## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/furniture-lab.git
   cd furniture-lab
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:5173`
